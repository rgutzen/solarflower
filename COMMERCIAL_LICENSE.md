## When do you need a commercial license?

The AGPL requires that if you use this software to provide a network service
(web app, API, SaaS product), you must publish all your source code — including
any modifications — under the AGPL. A commercial license removes this obligation.

You need a commercial license if you want to:

- Build a proprietary product or service on top of this codebase
- Integrate this software into a commercial offering without disclosing your source code
- Distribute a modified version without the AGPL's copyleft requirements
